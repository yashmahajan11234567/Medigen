# Functional Requirements

## Record Management
- Create Record
- Rename Record
- Delete Record
- Search Records
- Grid/List View

## Upload Types
### Prescription
Fields:
- Prescription Image/PDF (Required)
- Doctor Name
- Hospital/Clinic
- Doctor Specialty
- Consultation Date
- Follow-up Date
- Diagnosis
- Notes

### Pharmacy Bill
- Image/PDF upload only (V1)

### Other Document
- Images
- PDFs
- Blood Reports
- X-rays
- MRI/CT
- ECG
- Insurance documents
- Any healthcare document

## Connections
- Schedule (linked)
- Inventory (linked)
- Generic Searches (linked)

No OCR in Version 1.
